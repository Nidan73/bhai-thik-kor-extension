# Bhai Thik Kor Extension

A Grammarly-style browser extension companion for Bhai Thik Kor.

Goal: help users improve weak or vague prompts wherever they write, without forcing them to open the full web app.

Planned MVP:

- Normal Mode prompt improvement
- Guided Mode for unclear prompts
- Right-click selected text and improve it
- Keyboard shortcut for quick prompt fixing
- Floating suggestion card near active text fields
- Copy, replace, or insert improved prompts

The extension will call the existing Bhai Thik Kor backend instead of storing provider API keys in the browser.
